# RanzAI Design - Product Plan

RanzAI Design adalah AI co-designer untuk membantu kreator, UMKM, dan desainer membuat arahan desain secara cepat.

## MVP

- Landing page
- Tools page
- Design brief generator
- Layout suggestion
- Color direction
- Typography direction
- Copywriting helper
- Prompt visual helper
- Pricing page
- Basic dashboard

## Future

- Login Supabase
- Project history
- Brand kit
- Template library
- Credit system
- Payment webhook
- Admin dashboard
