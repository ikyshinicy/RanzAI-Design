# RanzAI Design

Starter repo untuk RanzAI Design, AI co-designer untuk bantu konsep desain, layout, warna, font, copywriting, dan prompt visual.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase ready
- OpenAI ready
- Payment ready structure

## Run Local

```bash
npm install
npm run dev
```

Buka:

```txt
http://localhost:3000
```

## Environment

Copy `.env.example` menjadi `.env.local`, lalu isi key yang dibutuhkan.

 
## MVP Feature

- Landing page
- Tools page
- Design brief generator local mode
- Dashboard starter
- Pricing starter
- Billing starter
- API design endpoint
